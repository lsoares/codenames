# Official Codenames Pictures

Drop the official picture-card images here — any `.jpg`, `.jpeg`, `.png`, `.webp`
or `.gif`. Each file becomes a possible card face; the "Official" deck draws 20
at random per game, so you need at least 20 files for it to appear (fewer and it
falls back to the word board).

Vite bundles and fingerprints whatever is in this folder at build time, so the
deck works locally and on the deployed site with no manifest to maintain — add or
remove files and they're picked up automatically.

These images are Czech Games Edition's copyrighted artwork. Keep this for your own
personal, non-commercial use.
